# Betting Bot

A Python-based betting bot that analyzes historical and current sports betting data to predict optimal odds placement.

## Features

- Data collection from open source and cheap historical data sources
- Real-time current data integration
- Machine learning models for odds prediction
- Decision-making engine for optimal betting placement

## Installation

1. Clone the repository
2. Create a virtual environment: `python -m venv venv`
3. Activate the virtual environment: `venv\Scripts\activate` (Windows)
4. Install dependencies: `pip install -r requirements.txt`

## Usage

Run the main bot: `python src/betting_bot.py`

## Project Structure

- `src/`: Main source code
- `data/`: Historical and current data files
- `models/`: Trained machine learning models
- `scripts/`: Data collection and utility scripts
